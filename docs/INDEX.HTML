Introduction

Leaf DOS or Leaf Diskette Operating System is a portable, 80x86 compatible operating system desinged to be runt on almost any 80's and 90's platform. Without any kind of restriction.

One of the main aims of Leaf DOS is to serve as a common DOS (nothing related to MS-DOS/FreeDOS) that can satisfy the needs of a operating system in older computers, while being minimalistic and incredibly small and fast.

Kernel

The kernel in Leaf DOS is the core of the operating system (thus a kernel!).

It is a monolitich kernel (all the stuff resides on the kernel memory space) , but it is slowly planed to switch to a microkernel (arbitrarily load drivers) so more functionality can be implemented.

The kernel has null security (real mode already does that), so antivirus or any kind of counter-security is not gonna be planned on a future (yet).

What the kernel can do

  • Load software

  • Provide a shell interface

  • Provide programs will call routines (API)

  • Load foreing software (see Compatibility with other OSes

What the kernel cannot do

  • Provide in-built shell programs/commands (all commands are programs!)

  • Provide a GUI

  • Have any form of security (it's not worth it on real-mode)

  • Load MS-DOS/FreeDOS programs (see Compatibility with other OSes

Collaborating

If you want to collaborate do a pull request on Leaf DOS github, or simply e-mail the source to me at Jesus Antonio Diaz. Attribution will be given obviously (in the file and in the Credits section of this page).

If you want your OS to be compatible with Leaf DOS feel free to do so.

If it's the other way around (Leaf DOS being compatible with your OS), feel free to e-mail me the source code/theory/resources or anything needed to be compatible with your OS.
